# User Guide Overview

Welcome to the Kadouri CRM User Guide! This guide is designed for non-technical users to help you get the most out of the system.

## What is Kadouri CRM?

Kadouri CRM is a complete order management system designed specifically for agricultural and produce sales businesses. It helps you:

- **Manage Your Customers** - Keep track of all your sellers and buyers
- **Create Orders** - Quickly create and manage orders with automatic calculations
- **Track Commissions** - Automatically calculate sales commissions
- **Generate Invoices** - Create professional PDF invoices instantly
- **View Reports** - See sales trends, customer history, and performance metrics
- **Sync with QuickBooks** - Automatically sync with your QuickBooks Online account

## Who Should Use This Guide?

This guide is perfect for:

- Sales representatives creating orders
- Account managers handling customer relationships
- Back office staff processing orders
- Accountants reviewing financial data
- Managers viewing reports and analytics

## Your Role Determines What You See

Kadouri CRM uses role-based access control, which means you only see the features you need:

### Admin
Full access to all features including user management and system settings

### Sales Representative
Access to accounts, orders, products, and commission reports

### Back Office
Access to accounts, orders, contracts, and email features

### Accountant
Access to invoices, financial reports, and view-only access to orders

Don't worry if you can't access certain features - it's by design to keep things simple!
